# minlinks

**Un solo enlace. Sin ruido. Sin sobreprecio.**

minlinks es un acortador de enlaces y agregador de presencia digital para creadores de contenido que valoran la simplicidad, la velocidad y el diseño de alto impacto.

## Principios
- **Minimalismo**: Solo lo que importa.
- **Velocidad**: De la idea a la publicación en segundos.
- **Impacto**: Estética premium por defecto.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui
- **Visuals**: Unicorn Studio integration
- **Theme**: Dark mode prioritized

## Estructura del Proyecto
- `/app`: Rutas y lógica de la aplicación.
- `/components`: Componentes UI y lógica de visualización (Unicorn Studio).
- `/lib`: Utilidades y configuraciones.

## Desarrollo
```bash
npm install
npm run dev
```
